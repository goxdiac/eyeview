# eyeview
Eyeview 1.0.0 launch now!
My Kreosus support link: https://kreosus.com/httpsgithubcomgoxdiac/about


A simple system monitor for Windows. Shows you what's running, how long it's been running, and what's using your network — all in one panel.
What it shows

Download and upload speed (live graph)
CPU and RAM usage (live graph)
How much data you've used this session
Open windows and how long they've been open
Which apps have active network connections
System uptime

Installation
-----
Download EyeView.exe from the Releases page and run it.
Or run from source:
bashpip install -r requirements.txt
python eyeview.py
-----
Requirements

Windows 10 or 11
Python 3.10+ (only if running from source)

Notes

Closes to system tray instead of exiting
Settings (colors, alerts, startup) accessible via the ⚙ button
Config is saved automatically
